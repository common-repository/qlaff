== Description ==

Plugin for adding Qlaff toplists, banners and games on wp pages.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress
2. Navigate to 'QLAFF Toplist' section in admin panel
3. Follow the instructions on specific pages
